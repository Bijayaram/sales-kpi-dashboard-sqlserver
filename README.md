# 📊 Sales Performance Dashboard | SQL + Power BI

## 📌 Project Overview

This project focuses on analyzing sales performance using SQL for data preparation and Power BI for visualization. The goal was to transform raw sales data into meaningful business insights by applying structured querying, KPI calculations, and interactive dashboard design.

The final result is a dynamic dashboard that highlights revenue trends, product performance, territory contributions, and year-over-year growth.

---

## 🎯 Objectives

* Perform data exploration and validation using SQL
* Build key business KPIs such as:

  * Total Revenue
  * Total Orders
  * Total Quantity Sold
  * Average Order Value
  * Year-over-Year Growth
* Create reusable SQL views for reporting
* Develop a professional Power BI dashboard
* Extract actionable business insights from the data

The workflow followed a structured analytical process:

3. **KPI Computation**

   * Revenue aggregation
   * Order volume calculation
   * Average Order Value
   * YoY growth calculation

4. **View Creation**

   * Connected views directly to Power BI
